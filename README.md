# 🌸 Aira — Anime AI Assistant

Aira is a locally hosted anime-style AI companion built with a Python Flask backend and a vanilla JS frontend. She supports both voice and text chat, responds with emotions, animates her expressions based on mood, and syncs her lips while speaking using the browser's built-in Text-to-Speech API. The UI has a soft magical pastel aesthetic with floating particles, glowing auras, and smooth animations.

Built with Python, Flask, HTML, CSS, and JavaScript. Voice input uses the Web Speech API (Chrome only).
